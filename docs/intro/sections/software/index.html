    Recommended software #

    Lectures #

    No particular software is required for the lectures.

    The exercises can be solved using web applications, for instance:

    • the Haskell playground and
    • the Java playground

    However, an IDE may be more comfortable.

    Assignments #

    For the assignments, you will need an IDE for Haskell and Java (possibly the same one).

    We provide elementary instructions below.

    Further details will be provided during the first lab (dedicated to software-related questions for this course).

    Haskell #

    Compiler and language server #

    You will need:

    • the Glasgow Haskell Compiler (GHC) and
    • The Haskell Language Server (HLS).

    These can be installed either:

    • via a package manager (Chocolatey on Windows, Homebrew on macOS, or your distribution’s package manager on Linux), or
    • with the GHCup installer.

    Warning. If you are using an Arch-based distribution, the GHCup installer is recommended.

    Editor #

    VSCode (or its open source version Code OSS) is a popular choice to develop in Haskell.

    The Haskell extension will let you benefit from the functionalities of HLS (autocompletion, go to definition, etc.) as well as automatic type inference. It can be installed from within VSCode (via the marketplace).

    You can also install the Haskell Run extension to run functions individually with a click.

    Java #

    Java Development Kit (JDK) #

    You will need a JDK (for instance OpenJDK) for Java 21 or later. There are many ways to install one:

    • Using a package manager (Chocolatey on Windows, Homebrew on macOS or your distribution’s package manager on Linux) is often recommended.
    • Alternatively, some editors (like VSCode on Windows and macOS) can come bundled with a JDK, whereas others (like IntelliJ IDEA) can help you download one.

    Maven #

    You will need Maven to run the assignment’s tests on your machine.

    The preferred way to install Maven is via a package manager (Chocolatey on Windows, Homebrew on macOS or your distribution’s package manager on Linux).

    Alternatively, you can download and install it manually (this may require setting up and/or updating environment variables).

    Editor #

    Popular editors/IDEs for Java include:

    • IntelliJ IDEA,
    • VSCode (or its open source version Code OSS),
    • Eclipse
    • NetBeans
